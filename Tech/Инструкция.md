Получение доступа к общему GitHub-репозиторию

1. Сообщить мне свой GitHub-логин (имя пользователя).
2. Я добавлю вас в коллабораторы
3. Принять приглашение, которое придет на электронную почту (которую вы использовали для регистрации) и появится на GitHub.
4. После принятия приглашения появится доступ к общему репозиторию.
5. Только после этого можно продолжать настройку синхронизации.

Установка Git

**Windows**

1. Перейти на страницу: [git-scm.com](https://git-scm.com/download/win)
2. Нажать на ссылку **64-bit Git for Windows Setup**.
3. Запустить скачанный файл `Git-…-64-bit.exe`.
4. В каждом окне установщика нажимать **Next**, не меняя никаких настроек.
5. В конце нажать **Install**.
6. После завершения установки нажать **Finish**.

Git полностью установлен.

**macOS**

1. Перейти на страницу: [git-scm.com/download/mac](https://git-scm.com/download/mac)
2. На странице откроется ссылка на загрузку `.dmg` файла Git для macOS.
3. Скачать установщик.
4. Открыть скачанный `.dmg`.
5. Запустить установочный файл **.pkg**.
6. В каждом окне нажимать **Continue**, затем **Install**.
7. При необходимости ввести пароль от macOS.
8. После завершения установки нажать **Close**.

Git полностью установлен.

Установка Git LFS (необходима для больших файлов)

**Windows**

1. Перейти на официальный сайт Git LFS: [git-lfs.com](https://git-lfs.com/)
2. Нажать кнопку **Download**.
3. Скачать установочный файл для Windows **.exe**.
4. Запустить скачанный файл.
5. В появившемся окне выбрать:
	+ **Next**
	+ Путь установки по умолчанию оставить без изменений
	+ **Install**
6. Дождаться завершения установки и нажать **Finish**.
7. После установки запустить **Командную строку Windows**:
- Нажать клавишу **Win + R**
- Ввести `cmd`
8. Ввести команду и нажать **Enter**:

```
git lfs install
```

9. Убедиться, что ошибок не появилось. Если ошибок нет — Git LFS установлен корректно.


**macOS**

1. Перейти на официальный сайт Git LFS: [git-lfs.com](https://git-lfs.com/)
2. Нажать **Download**.
3. Скачать установщик для macOS (**.pkg**).
4. Открыть скачанный файл.
5. В окнах установщика выбрать:
	+ **Continue**
	+ **Install**
6. Ввести пароль от macOS при необходимости.
7. Дождаться окончания установки.
8. Открыть **Terminal**:
- Нажать **Cmd + Space**
- Ввести `Terminal`
1. Ввести команду:

```
git lfs install
```

10. Убедиться, что ошибок нет.

Создание персонального токена GitHub для синхронизации Obsidian

(одинаково для Windows и macOS)

1. Перейти на страницу: [github.com](https://github.com/)
2. Убедиться, что выполнен вход в аккаунт.
3. В правом верхнем углу нажать на изображение профиля.
4. В появившемся меню выбрать **Settings**.
5. В левом меню пролистать список вниз.
6. Нажать **Developer settings**.
7. В открывшемся меню выбрать **Personal access tokens**.
8. Нажать **Tokens (classic)**.
9. Нажать кнопку **Generate new token**.
10. Выбрать **Generate new token (classic)**.
11. В поле **Note** можно ввести произвольное название, например: `monk`
12. В поле **Expiration** выбрать срок действия: `No expiration`.
13. В списке разрешений найти раздел **repo**.
14. Отметить галочку **repo**.
15. Пролистать страницу вниз.
16. Нажать кнопку **Generate token**.
17. Скопировать токен и сохранить в надежном месте.

Установка Obsidian

 **Windows**

1. Перейти на сайт: [obsidian.md](https://obsidian.md/)
2. Нажать кнопку **Download**.
3. Скачать версию **Windows (64-bit)**.
4. Запустить установочный файл.
5. Следовать инструкциям установщика, нажимая **Next**, затем **Install**, затем **Finish**.

**macOS**

1. Перейти на сайт: [obsidian.md](https://obsidian.md/)
2. Нажать **Download**.
3. Скачать версию **macOS (Intel или Apple Silicon)** — сайт предложит нужную автоматически.
4. Открыть скачанный файл **.dmg**.
5. Перетащить иконку Obsidian в папку **Applications**.
6. Открыть Obsidian через Spotlight:
- Нажать **Cmd + Space**
- Ввести `Obsidian`

Obsidian установлен.

Активация Obsidian

(одинаково для Windows и macOS)

1. Создать локальную папку на компьютере, где вы будите работать с хранилищем
2. Поместить в эту папку прикрепленную и распакованную папку `.obsidian`
3. Открыть приложение Obsidian
4. Выбрать пункт `Open folder as vault`
5. Указать путь к папке, которую вы создали раньше
6. При запуске выбрать "Доверять автору и активировать плагины"

Подключение к удаленному GitHub-репозиторию (клонирование vault)

 **Windows**

1. Находясь в Obsidian нажать `Ctrl + P`
**macOS**

1. Находясь в Obsidian нажать `Cmd + P`

**Далее одинаково**

2. Ввести `clone`, выбрать `Obsidian Git: Clone an existing remote repo`
3. Нажать Enter
4. Вставить ссылку без скобок `< >`:

```
https://<ВАШ СГЕНЕРИРОВАННЫЙ ТОКЕН>@github.com/kovalevegor/Monk_Vault.git
```

5. Нажать Enter
6. В появившемся окне выбора папки ввести `\`
7. Нажать Enter
8. Дождаться завершения клонирования
9. Obsidian автоматически откроет общую папку

Проверка работы автоматической синхронизации

Плагин Obsidian Git уже настроен заранее:

+ изменения сохраняются автоматически
+ обновления загружаются автоматически каждые 5 мин и при запуске obsidian
+ отправка изменений на сервер выполняется автоматически раз в 5 мин

Для проверки:

1. Внести любое изменение
2. Через несколько секунд появится значок синхронизации внизу
3. Через несколько минут изменения попадут в репозиторий
